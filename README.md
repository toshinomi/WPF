# 概要
【 ImageProcessing 】  
画像処理のアルゴリズム学習用に作成した WPF アプリケーションです。  
FWや画像処理ライブラリに頼らず、一からアルゴリズムを作成したい方の参考になればと思い作成しました。

開発環境：Visual Studio 2019  
言語：C#、Visual Basic.NET  

下記の機能を追加予定  
 ・処理時間を可視化できるグラフ機能  
 ・サーバサイドで画像処理ができる機能  
 ・画像処理中の進行状況の可視化機能  
 ・画像処理後の画像の保存機能  
 ・画像処理のアルゴリズム追加
